# Jianzi-Code 说明页设计记录

## 目标

写一份面向非程序背景读者的说明页，用于和音乐专业人士沟通 `Jianzi-Code` 的意义、价值和使用方式。

## 读者

- 古琴演奏者
- 古琴研究者
- 参与打谱与校勘的人
- 未来可能参与标注、校对和数据共建的人

## 写作原则

- 不预设读者理解 JSON、Schema、接口等术语
- 用“信息卡”“表单”“三层记录”这类直观比喻解释数据格式
- 重点讲清楚“为什么值得做”和“人怎么用”
- 不陷入实现细节

## 内容结构

1. `Jianzi-Code` 是什么
2. 它解决什么问题
3. 为什么对音乐专业的人有价值
4. 三层结构分别记录什么
5. 它不取代什么
6. 音乐专业的人如何参与使用
7. 为什么它值得成为独立项目
